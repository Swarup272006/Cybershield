# CyberShield

CyberShield is a web-based cybersecurity application that analyzes password strength,
provides intelligent password recommendations using AI, and detects suspicious attack
behavior such as brute-force attempts and abnormal input patterns.

## Core Features
- AI-based password strength analysis
- Intelligent password recommendations
- Threat attack detection
- Admin logs and security dashboard

## Tech Stack (Planned)
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask / FastAPI)
- Database: SQL
- AI: Rule-based + Generative AI
